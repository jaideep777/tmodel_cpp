Detailed Description

Everything about the plant's architecture, i.e. relationships among various physical dimensions.

Two variables completely define dimensional state of the plant: diameter and crown LAI. All other dimensions (height, crown area, leaf area, biomass, etc) are calculated from these two.

double plant::PlantArchitecture::dmass_dt_lai ( double &  dL_dt,
double  dmass_dt_max,
PlantTraits &  traits 
)

Rate of change of leaf mass due to change in LAI.

Parameters
dL_dtDesired LAI increment
dmass_dt_maxMaximum allowed biomass increment resulting from LAI change

Given the LAI change (dL_dt) and the maximum allowed mass increment (dmass_dt_max), this function calculates the mass increment (of leaves and fine roots) needed to achive the specified LAI increment. If this exceeds the maximum allowed mass increment, then mass increment is set to dmass_dt_max and dL_dt is revised accordingly. Complete coordination between fine roots and leaves is assumed. Thus, both leaves and fine roots need to increase for increasing LAI, and both are simultaneously shed if LAI decreases.
