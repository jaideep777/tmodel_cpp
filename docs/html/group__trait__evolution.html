Detailed Description

This is a collection of all classes and variables that are used in the evolutionary algorithm.

Set values for evolvable traits from vector.

Any changes to the algorithm or to the evolving traits will typically require changes in these variables.

Calculate growth rates of all species and update seed input.

Parameters
tCurrent time
dttimestep over which growth rate is to be calculated
SSolver

Species growth rate is defined from the seed perspective, i.e.,

\[r = \frac{1}{\Delta t}log\left(\frac{S_\text{out}}{S_\text{in}}\right),\]

where \(S\) is the seed rain (rate of seed production summed over all individuals of the species)
